These two divs are positioned next to one another using display: inline-block

I've used a reset ensure no elements should have any default margins or padding. (I did add a button margin to make it easier to read.) This space is actually a space character rendered just as it would be between two inine elements.

Wrong! Because, together, the two inline block elements will addd up to 100% of the available width, but that doesn't account for the space between.
